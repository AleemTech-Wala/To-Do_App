# Simple To-Do App

This is a simple To-Do List application built using HTML, CSS, and JavaScript. The project is designed to help users keep track of their daily tasks in an easy and organized way.

## Features
- Add new tasks
- Mark tasks as completed
- Remove tasks
- Responsive and user-friendly interface
- Built with pure HTML, CSS, and JavaScript (No frameworks)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was created to practice DOM manipulation, event handling, and basic front-end development concepts using vanilla JavaScript.
